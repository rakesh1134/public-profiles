Allow users to create and update their personal data (e.g., profile, posts, or projects) while making that data viewable to others.
Implemented using Ethereum smart contracts.

addprofile() function lets user create and update their personal data.

getmyprofile() function lets a user view his personal data

getprofile() function lets a user view other user's personal data (specify the user address as input)
